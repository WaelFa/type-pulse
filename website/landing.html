How It Works

Three steps. Zero friction.

Highlight any text, click, and the overlay appears. No setup, no accounts, no config pages.

Step 01

Select

Highlight any text on any webpage. A floating TypePulse button appears right next to your selection.

Step 02

Type

A clean overlay pops up with the selected text. Start typing — it highlights correct and incorrect characters as you go.

Step 03

Track

Watch your WPM, accuracy, timer, and errors update in real time. Get a clean session summary when you finish.

Features

Everything you need.
Nothing you don't.

  • Launch typing sessions from any selected text on any page
  • Context menu (right-click) or floating button to start
  • Real-time WPM, accuracy, timer, and error tracking
  • Clean session summary at the end of each session
  • Last session stats saved locally in chrome.storage
  • Toggle floating button on/off from the popup
  • No account needed — everything stays in your browser
  • 100% free and open-source

About

Built from a real struggle.

TypePulse was born from a personal struggle. I can read 3 paragraphs on a screen and forget what I even opened the tab for. Classic ADHD brain.

But I noticed something weird: the moment I start typing out the words I'm reading, my brain locks in. Full focus. I actually absorb every word instead of skimming and forgetting.

So instead of fighting it, I built an extension that leans into it. Now any article, documentation, or study material on the web becomes an active typing session — and everything just clicks.
